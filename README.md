# Presentation Generator

primary repository for generating presentations using LLMs.

## Tools

### [Claude Presentation Tool](./claude-presentation-tool/README.md)

A setup to generate high-quality HTML slides using a specialized system prompt for Claude.

## Quick Start from this repository

1. Go to `claude-presentation-tool`
2. Copy the `system_prompt.md`
3. Use it with Claude to generate amazing slides!
